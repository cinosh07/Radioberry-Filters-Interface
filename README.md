# Radioberry beta V3 Filters-Interface.

Radioberry project
https://github.com/pa3gsb/Radioberry-2.x

This project interface LPF, HPF and T/R Switch to the radiobery I/O over i2c.
The logic is arduino nano v3 based microcontroller.
